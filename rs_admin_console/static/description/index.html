Manual de Usuario - Uhuu API Tester

Descripción General:

Este módulo permite gestionar, ejecutar y registrar pruebas automáticas de endpoints API relacionados con clientes desde Odoo. Las pruebas pueden incluir autenticación (login) y operaciones GET/POST con estructura JSON, útiles para entornos como integraciones con Uhuu Chat.

Objetivo

Proveer un entorno dentro de Odoo para configurar endpoints de prueba por cliente, registrar su respuesta y estatus, ejecutar pruebas masivas y analizar los resultados obtenidos con ayuda de IA.

Guía de Uso

  • 1. Crear un entorno de pruebas en el menú "Entornos".
  • 2. Registrar uno o varios endpoints, indicando método, URL, parámetros, headers y si es un endpoint de tipo login.
  • 3. Asociar los endpoints a un cliente usando el campo Cliente asociado.
  • 4. Usar la vista de Consola por Cliente para ejecutar todas las pruebas del cliente con un solo clic.
  • 5. Visualizar el porcentaje de éxito, histórico de pruebas, y usar el botón de "Resultados" para filtrar los test realizados.
  • 6. Ejecutar análisis IA sobre respuestas fallidas para recibir recomendaciones de mejora.

Configuraciones Recomendadas

  • Clave OpenAI: configurar en "Parámetros del sistema" el valor rs_admin_console.openai_key.
  • Entorno por defecto: marcar uno de los entornos como predeterminado para las pruebas.
  • Tokens de autenticación: usar el endpoint de login para obtener tokens y reutilizarlos en los demás endpoints.
  • Relaciones: usar correctamente los campos de vinculación con cliente (partner_id) para que el resumen de consola funcione.
